# Feature 09 — UX / 미학

> Live floating, 자동 정렬, undo/redo, 레이아웃 저장, 테마.
> **예뻐야 한다.**

---

## Live Floating
- 각 노드가 제자리에서 미세하게 부유
- `sin/cos` 기반 오프셋, `requestAnimationFrame`
- 설정에서 on/off 토글
- 진폭: ~0.3px, 주기: 노드마다 다름 (자연스럽게)

## 자동 정렬
- Cytoscape 내장 레이아웃: `cose-bilkent` (물리 시뮬레이션)
- "Auto Layout" 버튼 → 현재 뷰 재정렬
- 드릴다운 전환 시 자동 실행

## 드릴다운 전환 애니메이션
- 모듈 더블클릭 → 줌인 + 페이드 → 내부 파일 그래프 등장
- Framer Motion `AnimatePresence` + Cytoscape `animate()`

## Undo / Redo
- 뷰 조작 이력: 노드 이동, 줌, 팬, 깊이 전환
- `Ctrl+Z` / `Ctrl+Shift+Z`
- 스택 크기 제한: 50

## 레이아웃 저장
- `.viber/layout.json`에 저장
- 깊이별(packages/modules/files) 각각 별도 저장
- 저장 항목: 노드 위치, 줌, 팬 오프셋

## 외부 패키지 토글
- "Show External" 버튼 → 패키지 노드 표시/숨김
- 숨길 때 관련 간선도 함께 숨김

## 테마
- 다크 모드 기본 (라이트 모드는 나중에)
- 색상 팔레트:
  ```
  bg-primary:   #0a0a0f
  bg-secondary: #12121a
  bg-surface:   #1a1a2e
  accent:       #e94560
  accent-2:     #533483
  accent-3:     #0f3460
  text:         #e0e0e0
  text-muted:   #8888aa
  success:      #4ecca3
  warning:      #ffc107
  ```
- 폰트: `JetBrains Mono` (코드), `Inter` (UI)
- 그림자, 글로우 효과 절제하되 핵심 포인트에 사용

## 반응형
- 최소 창 크기: 1024x600
- 사이드바 접기 가능
- Detail Panel 접기 가능

---

## API

```typescript
invoke('layout:save', {
  depth: 'packages' | 'modules' | 'files',
  positions: { [nodeId: string]: { x: number, y: number } },
  zoom: number,
  pan: { x: number, y: number },
}) → void

invoke('layout:load', { depth: 'packages' | 'modules' | 'files' })
  → { positions, zoom, pan } | null
```
